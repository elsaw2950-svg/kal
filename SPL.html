📘 Eliminasi Gaussian

Penyelesaian Sistem Persamaan Linier 3 Variabel dan 4 Variabel menggunakan metode Eliminasi Gaussian.

A. Sistem Persamaan Linier Tiga Variabel

1. Sistem Persamaan

x + y + z = 6
2x + 3y + z = 10
x - y + 2z = 5

2. Matriks Augmented

| 1  1  1 |  6 |
| 2  3  1 | 10 |
| 1 -1  2 |  5 |

3. Eliminasi Gaussian

Langkah 1

R2 → R2 - 2R1
R3 → R3 - R1

| 1  1  1 |  6 |
| 0  1 -1 | -2 |
| 0 -2  1 | -1 |

Langkah 2

R3 → R3 + 2R2

| 1  1  1 |  6 |
| 0  1 -1 | -2 |
| 0  0 -1 | -5 |

4. Substitusi Balik

-z = -5
z = 5

y - z = -2
y - 5 = -2
y = 3

x + y + z = 6
x + 3 + 5 = 6
x = -2

Jadi diperoleh: 

x = -2
y = 3
z = 5

B. Sistem Persamaan Linier Empat Variabel

1. Sistem Persamaan

x + y + z + w = 10
2x + y - z + w = 5
x - y + 2z + 3w = 15
3x + 2y + z - w = 8

2. Matriks Augmented

| 1  1  1  1 | 10 |
| 2  1 -1  1 |  5 |
| 1 -1  2  3 | 15 |
| 3  2  1 -1 |  8 |

3. Eliminasi Gaussian

Langkah 1

R2 → R2 - 2R1
R3 → R3 - R1
R4 → R4 - 3R1

| 1  1  1   1 | 10 |
| 0 -1 -3  -1 | -15 |
| 0 -2  1   2 | 5 |
| 0 -1 -2  -4 | -22 |

Langkah 2

R3 → R3 - 2R2
R4 → R4 - R2

| 1  1   1   1 | 10 |
| 0 -1  -3  -1 | -15 |
| 0  0   7   4 | 35 |
| 0  0   1  -3 | -7 |

Langkah 3

R4 → 7R4 - R3

| 1  1   1    1 | 10 |
| 0 -1  -3   -1 | -15 |
| 0  0   7    4 | 35 |
| 0  0   0  -25 | -84 |

4. Substitusi Balik

-25w = -84
w = 3.36

7z + 4w = 35
7z + 13.44 = 35
z = 3.08

-y - 3z - w = -15
y = 2.4

x + y + z + w = 10
x = 1.16

Jadi diperoleh:

x = 1.16
y = 2.4
z = 3.08
w = 3.36
