📘 Determinan dan Invers Matriks

Menghitung determinan matriks menggunakan metode ekspansi baris dan mencari invers matriks menggunakan metode adjoin.

A. Determinan Matriks dengan Ekspansi Baris

Rumus Umum

det(A) = Σ (-1)^(i+k) × aᵢₖ × Mᵢₖ

1. Matriks 2×2

A = | -7  -5 |
    |  1   4 |

Ekspansi Baris Pertama

det(A) =
(-1)^(1+1)(-7)M11 + (-1)^(1+2)(-5)M12

Minor

M11 = 4
M12 = 1

Perhitungan

= (+1)(-7)(4) + (-1)(-5)(1)
= -28 + 5
= -23

Hasil

det(A) = -23

2. Matriks 3×3

A = | 0   2  -3 |
    | 1  -2  -1 |
    | 0   0   1 |

Ekspansi Baris Pertama

det(A) =
(+1)(0)M11 + (-1)(2)M12 + (+1)(-3)M13

Minor M11

| -2  -1 |
|  0   1 |

= (-2)(১৷) - (-1)(0)
= -2

Karena dikali 0 maka hasil = 0

Minor M12

| 1  -1 |
| 0   1 |

= (1)(1) - (-1)(0)
= 1

(-1)(2)(1) = -2

Minor M13

| 1  -2 |
| 0   0 |

= (1)(0) - (-2)(0)
= 0

Karena hasil 0 maka:
(+1)(-3)(0) = 0

Hasil Akhir

det(A) = 0 - 2 + 0
det(A) = -2

3. Matriks 4×4

A =

|  1  -3   1   1 |
| -3   1   1   1 |
|  1   1  -3   1 |
|  1   1   1  -3 |

Ekspansi Baris Pertama

det(A) =
(+1)(1)M11 + (-1)(-3)M12
+ (+1)(1)M13 + (-1)(1)M14

Hasil Minor

M11 = 16
M12 = -16
M13 = 16
M14 = -16

Substitusi

det(A) =
(1)(16) + (3)(-16)
+ (1)(16) + (-1)(-16)

= 16 - 48 + 16 + 16
= 0

Hasil Akhir

det(A) = 0

B. Invers Matriks dengan Metode Adjoin

Rumus

adj(A)ᵢⱼ = (-1)^(i+j) Mⱼᵢ

A⁻¹ = (1 / det(A)) × adj(A)

4. Matriks 2×2

A =

| -7  -5 |
|  1   4 |

Determinan

det(A) =
(-7)(4) - (-5)(1)

= -28 + 5
= -23

Adjoin

adj(A) =

|  4   5 |
| -1  -7 |

Invers

A⁻¹ = (1 / -23)

|  4   5 |
| -1  -7 |

A⁻¹ =

| -4/23  -5/23 |
|  1/23   7/23 |

5. Matriks 3×3

A =

| 0   2  -3 |
| 1  -2  -1 |
| 0   0   1 |

Determinan

det(A) = -2

Matriks Kofaktor

C11 = -2
C12 = -1
C13 = 0

C21 = -2
C22 = 0
C23 = 0

C31 = -8
C32 = -3
C33 = -2

Adjoin

adj(A) =

| -2  -2  -8 |
| -1   0  -3 |
|  0   0  -2 |

Invers

A⁻¹ = (1 / -2) × adj(A)

A⁻¹ =

| 1    1    4 |
| 1/2  0   3/2 |
| 0    0    1 |

6. Matriks 4×4

A =

|  1  -3   1   1 |
| -3   1   1   1 |
|  1   1  -3   1 |
|  1   1   1  -3 |

Determinan

det(A) = 0

Kesimpulan

Karena det(A) = 0,
maka matriks tidak memiliki invers
(tidak invertible).
