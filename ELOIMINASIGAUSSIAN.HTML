Eliminasi Gaussian

Penyelesaian Sistem Persamaan Linier 3 dan 4 Variabel

1. Sistem Persamaan

x + y + z + w = 10
2x + y + z + w = 13
x + 2y + z + w = 14
x + y + 2z + w = 15

2. Matriks Augmented

B = matrix([
  [1, 1, 1, 1, 10],
  [2, 1, 1, 1, 13],
  [1, 2, 1, 1, 14],
  [1, 1, 2, 1, 15]
])
[ 1 1 1 1 | 10 ]
[ 2 1 1 1 | 13 ]
[ 1 2 1 1 | 14 ]
[ 1 1 2 1 | 15 ]

3. Eliminasi Gauss

Langkah 1

Menghilangkan elemen pada kolom pertama:

B.add_multiple_of_row(1, 0, -2)
B.add_multiple_of_row(2, 0, -1)
B.add_multiple_of_row(3, 0, -1)
[ 1 1 1 1 | 10 ]
[ 0 -1 -1 -1 | -7 ]
[ 0 1 0 0 | 4 ]
[ 0 0 1 0 | 5 ]

Langkah 2

Menghilangkan elemen pada kolom kedua:

[ 1 1 1 1 | 10 ]
[ 0 -1 -1 -1 | -7 ]
[ 0 0 -1 -1 | -3 ]
[ 0 0 0 -1 | 2 ]

4. Substitusi Balik

  • w = -2
  • z = 5
  • y = 4
  • x = 3
