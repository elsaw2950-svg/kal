3. Eliminasi Gauss (OBE)

Langkah 1: nolkan elemen di bawah pivot kolom 1

Operasi baris:

R₂ = R₂ − 2R₁

Penjelasan:

Pivot pertama ada di kolom 1 baris 1 (angka 1).

Di bawahnya (baris 2 kolom 1) ada angka 2.

Kita ingin membuat elemen tersebut menjadi 0.

Caranya dengan mengurangkan 2 kali baris 1 dari baris 2.

Langkah 2: nolkan elemen di bawah pivot kolom 2

Operasi baris:

R₃ = R₃ − 3R₂

Penjelasan:

Pivot kedua ada di kolom 2 baris 2 (angka 1).

Di bawahnya (baris 3 kolom 2) ada angka 3.

Kita membuat elemen tersebut menjadi 0 dengan mengurangkan 3 kali baris 2.

Langkah 3: nolkan elemen di bawah pivot kolom 3

Operasi baris:

R₄ = R₄ − 4R₃

Penjelasan:

Pivot ketiga ada di kolom 3 baris 3 (angka 1).

Di bawahnya (baris 4 kolom 3) ada angka 4.

Elemen tersebut dinolkan dengan mengurangkan 4 kali baris 3.

Langkah 4: nolkan elemen di bawah pivot kolom 4

Operasi baris:

R₅ = R₅ − 5R₄

Penjelasan:

Pivot keempat ada di kolom 4 baris 4 (angka 1).

Di bawahnya (baris 5 kolom 4) ada angka 5.

Elemen tersebut dibuat nol dengan mengurangkan 5 kali baris 4.

Kesimpulan

Eliminasi Gauss adalah metode efisien untuk menyelesaikan sistem persamaan linear (SPL) dengan mengubah matriks diperbesar menjadi bentuk eselon baris (segitiga atas) menggunakan operasi baris elementer. Proses ini diakhiri dengan substitusi balik untuk mendapatkan solusi, menjadikannya metode sistematis yang ampuh.
